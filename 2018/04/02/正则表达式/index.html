正则表达式

js正则表达式

// +表示匹配前面的子表达式一次或多次
// zo+ ”能匹配“zo”以及“zoo”，但不能匹配“z”
// zo ”能匹配“zo”，不能匹配“zoo”和“z
/^zo+$/i.test(‘zoo’)
/^zo$/i.test(‘zo’)
